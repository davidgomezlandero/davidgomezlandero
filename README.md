#### **ft_linear_regression** — Car Price Prediction
**Grade:** 125/100 ⭐

Implemented a linear regression algorithm **from scratch** to predict used car prices based on mileage.

**Stack:** Python 3.8+ | NumPy | Pandas | Matplotlib

**Key Features:**
- ✅ Manual gradient descent optimizer (200 epochs, configurable learning rate = 0.02)
- ✅ Data normalization using Z-score standardization for stable convergence
- ✅ Parameter denormalization for real-world predictions
- ✅ Three visualization types: raw data scatter, fitted line, loss evolution curve
- ✅ R² score calculation (R² = 0.72, explains 72% of price variance)
- ✅ Model persistence using CSV storage for theta parameters

**Dataset:** 24 car records | **Algorithm:** Gradient Descent + MSE Loss
